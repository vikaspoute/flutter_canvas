## 1.0.0 (2026-06-23)

### Added
- Infinite canvas with smooth pan, zoom, and rotation
- 12 built-in object types: Rect, Circle, Line, Arrow, Text, Image, SVG, Path, Polygon, StickyNote, Frame, Group
- Selection system: single, multi, shift, box/lasso select
- Resize and rotation handles
- Layer management: create, delete, lock, hide, rename, reorder, nest
- Undo/redo with Command Pattern and history compression
- Transaction support for grouped undo steps
- Smart guides with snap-to-object and snap-to-grid
- Grid system: square, dot, isometric
- JSON serialization with versioned schema
- PNG, JPEG, SVG export
- Collaboration interfaces (OT/CRDT architecture)
- Keyboard shortcuts (Ctrl+C/V/X/Z, Delete, arrows, zoom)
- Drawing tools: Pen, Shape, Text, StickyNote
- Minimap widget
- Dirty-region rendering optimization
- Viewport culling for performance
- Custom object registration for extensibility
- Example app demonstrating all features
- Comprehensive unit tests
